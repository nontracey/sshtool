# Changelog

All notable changes to SSH Tool will be documented in this file.

## [1.0.0] - 2026-04-24

### Added
- Initial release of SSH Tool for Android
- SSH host management (add, edit, delete, duplicate)
- Host grouping and organization
- Quick connection with one tap
- Connection history and statistics

### SSH Terminal Features
- Full PTY terminal support
- Password authentication
- SSH key authentication
- Real-time output display
- Command history navigation (↑/↓)
- Keyboard shortcuts (ESC, TAB, arrows, etc.)
- Auto-scroll with manual lock option
- Quick action buttons

### Settings & Customization
- Theme modes: Light, Dark, System default
- 10 beautiful color schemes (Blue, Indigo, Purple, Pink, Red, Orange, Yellow, Green, Teal, Cyan)
- UI font selection (Roboto, OpenSans, Lato, Montserrat, etc.)
- Terminal font customization (JetBrainsMono, FiraCode, Monaco, etc.)
- Terminal font size adjustment (8-32sp)
- Line height customization (1.0-3.0)
- Cursor styles (Block, Underline, Bar)
- Bell sound and vibration options
- Keep screen on toggle
- Scrollback lines configuration (1000-50000)
- Connection timeout settings

### Technical Features
- Material Design 3 UI
- Responsive layout for all screen sizes
- Hardware keyboard support
- Optimized performance (60fps animations)
- Small APK size (8-15MB)
- Fast startup time
- Efficient state management with Riverpod
- Fast data persistence with Hive database
- ProGuard optimization for release builds

### UI Components
- Beautiful host cards with color coding
- Search and filter functionality
- Empty states with helpful guidance
- Connection status indicators
- Quick actions bar
- Form validation
- Loading states
- Error handling

### Platform Support
- Android 5.0+ (API level 21)
- Supports both ARM and ARM64 architectures
- Portrait and landscape orientations
- Tablet and phone optimized

### Security
- Secure credential storage
- SSH key support
- Encrypted data persistence
- No analytics or tracking

### Known Limitations
- SFTP file transfer not yet implemented
- Port forwarding not yet available
- No cloud sync (planned for future release)
- No multi-tab sessions (planned for future release)
